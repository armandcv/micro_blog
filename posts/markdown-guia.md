Markdown es un lenguaje de marcado ligero que puedes aprender en minutos. Aquí un resumen de la sintaxis más útil.

## Encabezados

```
# H1 — Título principal
## H2 — Sección
### H3 — Subsección
```

## Texto

| Sintaxis | Resultado |
|----------|-----------|
| `**negrita**` | **negrita** |
| `*cursiva*` | *cursiva* |
| `~~tachado~~` | ~~tachado~~ |
| `` `código` `` | `código` |

## Listas

**Sin orden:**
```
- Elemento uno
- Elemento dos
  - Sub-elemento
```

**Con orden:**
```
1. Primero
2. Segundo
3. Tercero
```

## Links e imágenes

```markdown
[Texto del link](https://ejemplo.com)
![Texto alternativo](imagen.png)
```

## Citas

```
> Esta es una cita destacada.
> Puede ocupar varias líneas.
```

> Esta es una cita destacada.
> Puede ocupar varias líneas.

## Bloques de código

Usa tres backticks y el nombre del lenguaje:

````
```python
def hola(nombre):
    print(f"Hola, {nombre}!")
```
````

```python
def hola(nombre):
    print(f"Hola, {nombre}!")
```

## Separador horizontal

```
---
```

---

¡Eso es todo lo esencial! Con esto puedes escribir posts completos y bien formateados.
